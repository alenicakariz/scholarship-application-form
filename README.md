# College Scholarship Application Form

## Site Link
- https://alenicakariz.github.io/scholarship-application-form/

## About the Project
This is a simple College Scholarship Application Form built as a hands-on exercise in **HTML form** development and **CSS** styling. The form simulates a real-world scholarship application interface, covering multiple sections that a typical application would include. 

## Importance Notice
This form **does not**:
- Submit data to any server or database
- Store any personal information
- Represent a real scholarship opportunity
